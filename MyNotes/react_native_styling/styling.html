Q
what is flex box and what is this for?
A
flex box gives a way for parent to specify the layout of its children.
We will need to use flexDirection, alignItems and justifyContent to achieve the right layout.

flexDirection
=>determine the primary axis of its layout,
=>Should the children be organized horizontally or vertically? the default setting is to vertically.

Possible value:
row: to organize child components horizontally.
column: to organize child components vertically.
Ex:
<View style={{flex: 1, flexDirection: 'row'}}>
justifyContent
determines the distribution of children along the primary axis.
Available options are
flex-start: children will be rendered at the start of the parent component
center:children will be rendered at the center of the parent component
flex-end: children will be rendered at the bottom of the parent component
space-around: children will be rendered spaced evenly
space-between:

determines the alignment of children along thesecondary axis

Q

A

ex:

header:{
    flex:1
  },

The flex box 1 means to take as much space as it can be.

Q

A
By giving a border to the flex box:
We can have a function:
border(color){

    }
  }

Q
Is it recommended to set the size of component by setting fixed height?
A
=>Setting up the layout, position the component, is not recommended to use fixed height. It is better to use flex box
=>for specific component, like a button, or a picture, it is okay to use fixed height.
