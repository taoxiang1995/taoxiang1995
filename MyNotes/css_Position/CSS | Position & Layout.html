  • Text layout
  • Box Layout
Layout between Boxes (visual formatting):
  • Normal Flow layout:

  • out of normal flow layout:

Q
What is the difference between inline element and block element?
A
inline element and block element are two basic categories of elements in HTML.

inline
Think it as a box that acts like text
=>Flow along with text content, will not clear previous content to start  a new line
=>ignore top and button margin setting.
=>can set any paddings
=>ignore height and width
=>if floated to left or to right, will become a block element.
Ex:
<a><span><b><em><i>
block
=>If no width set, it will try to fill entire parent container
=>If no height is set, will expand just enough to hold its children
=>can have margin and padding
=>By default, will be placed below previous elements
Examples of Block Elements:
<p><div><form><header><nav><ul><li><h1>
replaced element
=>Replaced elements are element whose content are from outside source:
<img><iframe><button><input><textarea>

Q
Can we put a block element inside of a inline element?
A
No we can not but with one exception:
We can put <a> which is a block element inside of any element, even a inline element.

Q
What is BOX model?
A
=>Every HTML element can be considered as a box model (some has margin, some doesn’t)
=>we can specify size, radius, style and color for border.
'border-top', 'border-right', 'border-bottom', 'border-left'
Value:   [ <border-width> || <border-style> || <'border-top-color'> ] 


Q
What are the positioning properties and how to use them?
A

static
=>this is default for every page element.
=>it means element will be positioned as it normally would, 
=>change top, left, button, right will not affect it.
relative
=>positioned relative to itself.
=>limit the scope of absolutely positioned child element.
absolute
=>set the element position relative to the next element with relative or absolute positioning. 
=>will be removed from normal flow. can’t affect other element, other element can’t affect it. 
fixed
=>set the element relative to the screen.

     }

<div class="wrap">

</div>

Q
How to center a Box element?
A
[Basic idea]
There are two ways to center a box element.
(1) using padding
(2) using margin

Using padding to center the box element:
<div class="wrap">
     <div class="content"></div>
</div>

<style>
     @wrapWidth:400px;
     .wrap{
          margin-left: auto;
          margin-right: auto;
          margin-top: 20px;
          width: @wrapWidth;
          height: @wrapWidth;
     }

     .content{
          @contentWidth: 100px;
          height: @contentWidth;
          width: @contentWidth;
          padding: (@wrapWidth-@contentWidth)/2
     }
</style>
Using margin to center the box element:
<div class="wrap">
     <div class="content"></div>
</div>

<style>
     @wrapWidth: 400px;
     @contentWidth: 100px;
     .wrap{
          height: @wrapWidth;
          width: 100%;
          overflow: hidden; /* prevent margin collapsing */
     }

     .content{
          margin-left: auto;
          margin-right: auto;
          margin-top: (@wrapWidth - @contentWidth)/2;
          width: 100px;
          height: @contentWidth;
     }
</style>

Q
What is margin collapsing rule?
A
Top and Buttom margin of blocks are sometimes merged into a single margin whose size is the larger one of the margin. 

Q
How to prevent margin collapsing?
A
Way1: set parent overflow to hidden or auto:    
.parent { overflow: auto; }
.parent { overflow: hidden; }
Way2: adding a border:
.ele{
     border: 1px
}
Way3: change padding instead of margin
