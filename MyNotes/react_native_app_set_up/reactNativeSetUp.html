Q
What is the basic boilerplate for react-native iOS component?
A
Basically, a component file has 4 distinct sections:
  1. importing: import the libraries we need
  2. Defining: create our component
  3. Styling: style our component
  4. Exporting: make this code available else where.
import libraries

text is like <p>
view is like <div>
import React, {
  AppRegistry,
  Component,
  StyleSheet,
  Text,
  View
} from 'react-native';
define the component class
class Weekdays extends Component{
  render(){
    return(
      <View style={styles.container}> 
        <Text>
          Days of the week:
        </Text>
      </View>
    );
  }
}
style the component
const styles = StyleSheet.create({
  container:{
    flex:1,
    justifyContent: 'flex-end', //move element height wise
    alignItems:'center' //move stuff width wise
  }
});
render component to screen
'weekdays' is the name of application
()=>Weekdays
defines a function that returns defined class component Weekdays.
AppRegistry.registerComponent('weekdays', () => Weekdays);

class UselessTextInput extends Component {

=>scrollView is a generic container that can hosts multiple components and views.

ListView
Q
What is ListView and why do we need it?
A
The ListView component displays a vertically scrolling list of changing, but similarly structured, data.
ListView requires two props:
=>dataSource: it is the source of information of the lists
=>renderRow: takes in one item from the source and return a component (JSX) to render.

import React, { Component } from 'react';
import { AppRegistry, ListView, Text, View } from 'react-native';

class ListViewBasics extends Component {
  // Initialize the hardcoded data
  constructor(props) {
    super(props);
    const ds = new ListView.DataSource({rowHasChanged: (r1, r2) => r1 !== r2});
    this.state = {
      dataSource: ds.cloneWithRows([
        'John', 'Joel', 'James', 'Jimmy', 'Jackson', 'Jillian', 'Julie', 'Devin'
      ])
    };
  }
  render() {
    return (
      <View style={{paddingTop: 22}}>
        <ListView
          dataSource={this.state.dataSource}
          renderRow={(rowData) => <Text>{rowData}</Text>}
        />
      </View>
    );
  }
}

AsyncStorage
Q
What is AsyncStorage?
A
AsyncStorage is a simple, unencrypted, asynchronous, persistent, key-value storage system that is global to the app. It should be used instead of LocalStorage.
