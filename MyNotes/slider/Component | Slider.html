Key features explanation:

(1) two sliders, one at bottom one at middle
(2) able to slide new background image in with animation.
(3) each slide contains text slider
(4) slider toggles have animation when switching

Q
How to make animation when toggle switch from one point to another?
A
[Basic Idea]
(1) create a bigger toggle on top of current toggle
(2) create two variables to store the current position and target position of toggle
(3) when toggle clicked, use .animate( ) function to move toggle from current position to target position, at the same time, change it size
=>middle position => target position => resize
$('#selector').animate({ width: '20px', height: '5px', left: CenterPxs + 'px', top: '2px' }, 200, function () {
  $('#selector').animate({ width: '18px', height: '18px', left: EndPxs + 'px', top: '-4px' }, 100, function () {
  $('#selector').animate({ width: '14px', height: '14px', top: '-2px' }, 100) }) })
}

Q
How to implement the slides feature? when toggle clicked, move from current scene to another scene?
A
[Basic Idea]
Have two div, one for current scene, one for new scene. 
when toggle clicked, position the new scene to the left or right of the screen (depending on which direction we want the scene to slide)

Steps:
(1) position nextSlide to the right (or left) of the screen and show it.
  $('#nextVideo').css('right', screenWidth + 'px');
        $('#nextVideo').show();
(2) move nextSlide to the center of screen
$('#nextVideo').animate({ right: '0px' }, time, 'linear');
(3) move curSlide left (or right) a little bit (to create a feel of movement)
$('#curVideo').animate({ right: '-100px' }, time, 'linear', function () { setCurSeason(id); showSliderText(id); });
      }
(4) make nextSlide currentSlide, make currentSlide nextSlide and change their Z-index
  $('#curVideo').attr('id', 'nextVideoNew');
  $('#nextVideo').attr('id', 'curVideo');
  $('#nextVideoNew').attr('id', 'nextVideo');
  $('#nextVideo').hide();
  $('#curVideo').css('z-index', '0');
  $('#nextVideo').css('z-index', '1');
