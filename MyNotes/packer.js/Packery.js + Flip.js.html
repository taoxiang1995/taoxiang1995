Goal: Use packery and Flip.js to create grid card effect.

Q
What is Packery.js and Flip.js and where to find them?
A
Packers.js is used to create grid effect, while Flip.js is used to create flipping card effect.
Where:
Flip.js: https://nnattawat.github.io/flip/
Packers: http://packery.metafizzy.co/

Q
How to integrate Packery with Flip?
A
<div class="grid">
            <div class="grid-sizer"></div>

            <div id="card" class="grid-item noColor">

                <div class="front  skyColor react">
                  <img src="img/react.png" style="width:100%; height:100%;"></img>
                </div>

                <div class="back  tealColor react">

                </div>

            </div>

Notice that we need to set the background color of card to transparent.
Ex:
.noColor{
  background: rgba(255,0,0,0)
}
