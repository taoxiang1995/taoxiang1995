Q
What is provided by Redux-Form when you connect your form component to redux?
A
(Why) Do we need to use the property provided by redux-form?
we need to use the methods provided by redux form to submit the form,
handling the validation and display the error info and more.
(What) Is provided?
=>valid / invalid indicating if the form input is valid or not.
=>reset( ) to reset the form state.
=>handleSubmit( ) to submit the form input to the state. 

Q
How to use redux-form?
A
(1) Install redux-form
npm install --save redux-form
(2) Give redux-form reducer to Redux in the index.js file of reducer directory. 
import { reducer as formReducer } from 'redux-form'

const reducers = {
  // ... your other reducers here ...
  form: formReducer     // <---- Mounted at 'form'
}
(3) In the form component, import redux-form module
import { Field, reduxForm } from 'redux-form'
(4) Create form component and export needed form methods and property.
const SimpleForm = (props) => {
     const { handleSubmit, pristine, reset, submitting } = props;
}
(5) Use Field instead of input to create input element. 
    //normal input 
     <div>
        <label>First Name</label>
        <div>
          <Field name="firstName" component="input" type="text" placeholder="First Name"/>
        </div>
      </div>

        // radios
     <div>
        <label>Sex</label>
        <div>
          <label><Field name="sex" component="input" type="radio" value="male"/> Male</label>
          <label><Field name="sex" component="input" type="radio" value="female"/> Female</label>
        </div>
      </div>

           //select options
     <div>
        <label>Favorite Color</label>
        <div>
          <Field name="favoriteColor" component="select">
            <option></option>
            <option value="ff0000">Red</option>
            <option value="00ff00">Green</option>
            <option value="0000ff">Blue</option>
          </Field>
        </div>
      </div>
(6) register handleSubmit method.
<form onSubmit={handleSubmit}>
(7) Decorated with reduxForm and export it.
export default reduxForm({
  form: 'simple'  // a unique identifier for this form
})(SimpleForm)

Q
How does redux-form handle submission?
A
(How & when) does redux-form update user input to redux state?
If the form component is decorated by redux, then user input will be updated automatically to the state.

=>Redux will provide decorated component an handleSubmit method. 
=>handleSubmit should be called when submit the form or click the submit button:
<form onSubmit={handleSubmit(this.save.bind(this))}>
OR
<button onClick={handleSubmit(this.save.bind(this))}></button>

(what) does handleSubmit function do?
It does 2 things:
(1) run the input validation
(2) call this.props.onSubmit(data)  which is an user defined method, which will takes in an argument data, which contains user input state.

How to provide this.props.onSubmit?
there are 2 ways:

1. pass it a function to call

<button onClick={handleSubmit(data => {
  // do something with data. validation will have been called at this point,
  // so you know the data is valid
})}>Submit</button>

2. pass in such a function as the onSubmit prop to your decorated component


<MyDecoratedForm onSubmit={data => {
  // do something with data. validation will have been called at this point,
  // so you know the data is valid
}}/>
