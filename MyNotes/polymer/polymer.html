Q
What features does polymer provide?
A
Register element
What: register an element associates with tag name and prototype
Why we need this: 
(1) we can use it as a dom element in other html page
(2) we can add property and events to this customer element just like native dom element.
How to register an element:

<link rel="import" href="https://polygit2.appspot.com/components/polymer/polymer.html">
 
<script>
// register a new element called proto-element
  Polymer({
    is: "proto-element",
    // add a callback to the element's prototype ready:
    function() {
      this.textContent = "I'm a proto-element. Check out my prototype!"
     }
});
 
</script>
local dom template
What: add native dom elements to polymer web components
Why: we need to use native dom to implement our own UI
How:
<link rel="import" href="https://polygit2.appspot.com/components/polymer/polymer.html">
<dom-module id="dom-element">
  <template>
    <p>I'm a DOM element. This is my local DOM!</p>
  </template>
  <script>
    Polymer({ is: "dom-element" });
  </script>
</dom-module>
compose local dom
What: we can write parent dom for local doms
Why: sometimes we just needs to write a wrap component, say, a photo frame, in this case we do not know what will be the child component yet. 

How:
<link rel="import" href="https://polygit2.appspot.com/components/polymer/polymer.html">
<dom-module id="picture-frame">
  <template>
    <!-- scoped CSS for this element -->
    <style>
      div { display: inline-block; background-color: #ccc; border-radius: 8px; padding: 4px; }
    </style>
    <div>
      <!-- any children are rendered here -->
      <content></content>
    </div>
  </template>
  <script>
    Polymer({ is: "picture-frame", });
  </script>
</dom-module>
property binding
What: Property is like native mom’s property where we can pass in value to the web component. 
Why: Sometimes, we need to pass in some value to a web component, and web component use this passed in value to render. 
How:
<link rel="import"  href="https://polygit2.appspot.com/components/polymer/polymer.html">

<dom-module id="configurable-name-tag">

  <template>
    <!-- bind to the "owner" property -->
    This is <b>{{owner}}</b>'s configurable-name-tag element.
  </template>
  <script>
    Polymer({
      is: "configurable-name-tag",
      properties: {
        // declare the owner property
        owner: {
          type: String,
          value: "Daniel"
        }
      }
    });
  </script>
</dom-module>
lifecycle callbacks

data binding
